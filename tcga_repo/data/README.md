# Data

This folder contains TCGA clinical data. Files are not tracked due to size.

## Source

Data is publicly available from the [GDC Data Portal](https://portal.gdc.cancer.gov/).

Download clinical data for any TCGA project (e.g. TCGA-BRCA, TCGA-LUAD) and place the files here before running the notebooks.

## Expected files

| File | Description |
|------|-------------|
| `clinical.tsv` | Patient-level clinical data (age, survival, diagnosis, treatment) |
