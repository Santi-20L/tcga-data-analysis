# Data

This folder contains input data files required to run the analysis notebooks.Some files are not tracked in this repository due to size constraints.

## Required files

| File | Source | Used in | Description |
|------|--------|---------|-------------|
| `clinical.tsv` | GDC Data Portal | Notebooks 01–03 | Raw TCGA-BRCA clinical data (patient demographics, diagnosis, survival) |
| `clinical_clean.csv` | Generated by notebook 01 | Notebooks 02–03 | Cleaned and preprocessed clinical dataset |
| `TCGA.BRCA.sampleMap_HiSeqV2` | UCSC Xena | Notebook 04 | Gene expression matrix (20,530 genes × 1,218 samples, log2-normalized) |
| `TCGA-BRCA.survival.tsv` (or equivalent phenotype file) | UCSC Xena | Notebook 04 | Patient phenotype and survival annotations |

---

## How to download

### Clinical data (GDC Portal)
1. Go to [https://portal.gdc.cancer.gov/](https://portal.gdc.cancer.gov/)
2. Search for **TCGA-BRCA**
3. Navigate to the **Clinical** tab
4. Download as **TSV** and place in this folder as `clinical.tsv`

### Gene expression + phenotype (UCSC Xena)
1. Go to [https://xenabrowser.net/datapages/](https://xenabrowser.net/datapages/)
2. Select **TCGA Breast Cancer (BRCA)**
3. Download:
   - **Gene Expression RNAseq** → `IlluminaHiSeq` → file: `TCGA.BRCA.sampleMap_HiSeqV2`
   - **Phenotype** → `TCGA.BRCA.sampleMap/BRCA_clinicalMatrix` or equivalent survival file
4. Place both files in this folder

---

## Notes

- `clinical_clean.csv` is generated automatically by running `01_data_cleaning.ipynb` — do not download separately
- Gene expression values are log2(x+1) normalized RSEM estimates
- All files should be placed directly in this `data/` folder before running the notebooks
